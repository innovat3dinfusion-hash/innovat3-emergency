Install in Titan Mail

  1. Click Copy HTML for Titan Mail above
  2. Open mail.titan.email — sign in with info@innovat3.co.za
  3. Gear icon → Settings → Signatures → + Create New Signature
  4. Name it Innovat3 Main
  5. Click the HTML button → Ctrl+A → Ctrl+V → OK → Save
  6. Set as default for new emails AND replies
